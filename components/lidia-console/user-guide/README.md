# Lidia Console User Guide

**Lidia Console Kullanıcı Kılavuzu**’na hoş geldiniz. Bu dokümantasyon, Lidia Marketplace Console’un tüm yeteneklerini etkili ve verimli şekilde kullanabilmeniz için hazırlanmıştır. Kılavuz boyunca, sistemin temel yapı taşlarını, yönetim ekranlarını ve operasyonel akışları adım adım keşfedecek; günlük operasyonlardan gelişmiş yönetsel ayarlara kadar ihtiyaç duyabileceğiniz tüm bilgilere erişebileceksiniz.

İster Lidia Console’u ilk kez kullanan bir kullanıcı olun, ister platformu yöneten deneyimli bir yönetici, bu rehber Lidia Console deneyiminizi en verimli seviyeye taşımayı amaçlar.
